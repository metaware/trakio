# trakio

trakio is a library for tracking events and sending profile updates to http://trak.io from your ruby applications. 

## Installation

  $ gem install trakio

## Getting Started

```Ruby
require 'trackio'
tracker = TrakIO::Tracker.new(YOUR_TOKEN)
```
